# 排版结构：  
- 使用  # / ##  分级标题划分模块，用  ---  分隔线区分不同内容区域​
- 内容展示：通过  - / •  无序列表（搭配 emoji）清晰罗列信息​
- 交互元素：用  [显示文字](链接地址)  插入超链接，用  ![替代文字](图片地址)  插入静态图片​
- 格式强调：用  **加粗**  突出重点文本，用 ``` 代码块包裹命令/代码片段
