4.要加粗文本在文本前后各写两个星号即可。如：**Markdown**  
5.要使文字是斜体部分，则在需要进行斜体操作的文字前后各写一个星号即可。如：*我爱学习*   
6.三个星号放在文字前后即可进行加粗加斜体的同时操作。如：***我爱学习***  
7.删除线使用：在文字前后两端各写两次波浪符号。如：~~删除~~  
8.分割线：三个及以上的下划线或者星号单独写在某一行（中间不能插入别的字符） ，即可在该行有分割线的效果，使文本看起来更清晰明了。如下：

14.代码块：在所写代码的第一行的上一行和最后一行的下一行都加上三个`符号，若还想申明写的是什么语言类型的代码还可写上该语言的名字。如：  
```  c++
int main() {
    cout << "Hello, World!" << endl;
    return 0;
}  
```    
若要调用某方法的话把该方法写出来且在前后各加上一个符号即可,如：`equals()`
